# Oh!! Mine サイト運用マニュアル

このドキュメントは、商品を追加・編集するための手順書です。

## 📁 1. 画像の準備
撮影した商品画像を、以下のフォルダに保存してください。
ファイル名は半角英数字（`new-bag-01.jpg` など）推奨です。

**保存場所:**
`public/images/products/`
  ├─ `bag/` ...... バッグ、サコッシュ
  ├─ `wallet/` ... 財布、コインケース
  ├─ `case/` ..... キーケース、マルチケース
  └─ `accessory/`  小物、サシェなど

---

## 📝 2. 商品データの追加
`src/data/products.js` をエディタで開きます。
以下のようなブロックをコピーして、一番上（または適切な位置）に追加してください。

```javascript
  {
    id: 101, // 【重要】他の商品と被らない新しい数字にする
    name: '新しい商品の名前',
    price: 5800, // カンマなしの数字
    priceStr: '¥5,800', // 表示用
    category: 'Bag', // Bag, Wallet, Case, Accessory のいずれか
    tagline: '一覧画面に出るキャッチコピー',
    description: '商品の詳しい説明文。\n改行したいときはこうやって書きます。',
    details: [
      '素材: デニム',
      'サイズ: 20cm x 15cm',
      '機能: 内ポケットあり'
    ],
    care: '手洗い推奨です。',
    reviews: [], // レビューがなければ空のまま []
    
    // メイン画像（1枚目）
    image: '/images/products/bag/my-new-bag.jpg', 
    
    // スワイプ用画像（2枚目以降があれば）
    images: [
      '/images/products/bag/my-new-bag.jpg',
      '/images/products/bag/my-new-bag-detail.jpg'
    ],
    
    // カードの大きさ（通常はこれでOK）
    gridClass: 'md:col-span-1 md:row-span-1', 
  },
```

---

## 🚀 3. 公開（デプロイ）
変更を保存したら、Gitでプッシュしてください。

```bash
git add .
git commit -m "商品追加: 商品名"
git push
```

Vercelが自動的に検知して、数分後にサイトが更新されます。

---

## 📖 4. ストーリー（Aboutページ）の編集
ブランドの想いを変更したい場合は、`src/pages/About.jsx` を編集します。

*   **文章の変更:** 日本語のテキスト部分を書き換えてください。
*   **改行:** 文章の中で改行したい場合は `<br/>` と書きます。
*   **画像の変更:** `src="..."` の中のURLを、新しい画像のパス（`/images/...`）に変更します。

---

## 🆘 困ったときは
*   **画像が出ない:** パス（`/images/...`）が間違っていませんか？
*   **エラーになる:** カンマ `,` の付け忘れや、カッコ `}` の閉じ忘れがないか確認してください。
